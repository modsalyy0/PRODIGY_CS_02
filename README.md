# PRODIGY CS Task-02
## Image Encryption using Pixel Manipulation
This project encrypts and decrypts images by modifying pixel values.
## Features
- Encrypt image using key
- Decrypt image using same key
- Simple pixel manipulation logic
## Technologies
- Python
- Pillow library
## How to Run
1. Install Pillow: pip install pillow
2. Run the script
3. Provide image path and key
## Output
- encrypted.png
- decrypted.png
